## Project description / pitch
```
Problem: agents constantly present data to humans — and LLMs hallucinate statistics.
When an agent's report is wrong, trust in the whole agent economy erodes.

Plotline is an Agentic Service Provider that makes hallucinated numbers structurally
impossible while making data genuinely enjoyable to read:

1. Facts before prose — a deterministic TypeScript statistics engine computes a ranked
   fact ledger from raw CSV: OLS trends, binary-segmentation changepoints, MAD outliers,
   Pearson correlations, streaks, seasonality, category leaders, concentration.
2. The numeric audit — the AI narrator may only write around ledger facts; every number
   in its prose is extracted and verified against the ledger (0.05 abs / 1.5% rel
   tolerance). Failing scenes are rewritten deterministically. The audit score is
   printed on every story.
3. Cinematic output — one self-contained scrollytelling HTML page: charts that draw
   themselves, count-up hero stat, 3 themes, 3 tones, the full fact-ledger appendix,
   zero external requests.

Agent-native: 5 MCP tools over stateless Streamable HTTP (create_story, analyze_csv,
get_story, list_themes, get_pricing) with structured outputs. Monetized with x402:
free daily quota (HTTP 200 direct results, A2MCP-compliant), then $0.20 USDT0 per
story on X Layer (spec-correct 402 + PAYMENT-REQUIRED challenge).

Quality: 30 automated tests (stats known-values, audit rejection of fabricated numbers,
XSS escaping, end-to-end pipeline), strict TypeScript, Docker deploy, no database,
LLM-optional with deterministic fallback — the service cannot fail its review because
an upstream is down.

Why it wins: everyone else built plumbing for crypto agents. Plotline is what agents
USE the plumbing for — the moment an agent turns data into something a human trusts
and enjoys. It's the missing presentation layer of the agent economy.
```
